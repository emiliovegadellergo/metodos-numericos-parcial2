Sobre el método — Newton-Raphson

El método de Newton-Raphson es un algoritmo iterativo para encontrar raíces de funciones. Parte de una estimación inicial x₀ y, en cada paso, aproxima la función con su recta tangente para obtener una mejor estimación.

Fórmula de iteración:

x_{n+1} = x_n − f(x_n) / f′(x_n)

Cada nueva aproximación se obtiene restando al valor actual el cociente entre la función y su derivada evaluadas en ese punto. Geométricamente, es la intersección de la recta tangente con el eje x.

Ventajas y cuándo usarlo:

  • Convergencia cuadrática (muy rápida)
  • Pocas iteraciones necesarias
  • Ideal cuando f′(x) es fácil de calcular
  • Preciso cerca de la raíz
Procedimiento paso a paso
  1. Definir la función y su derivada: f(x) = 5eˣ − 2x − 10 f′(x) = 5eˣ − 2
  2. Evaluar en el valor inicial x₀ = 0: f(0) = 5e⁰ − 2(0) − 10 = 5 − 10 = −5.00000 f′(0) = 5e⁰ − 2 = 5 − 2 = 3.00000
  3. Aplicar la fórmula de Newton-Raphson para obtener x₁: x₁ = x₀ − f(x₀)/f′(x₀) = 0 − (−5)/3 = 0 + 1.66667
    x₁ = 1.66667  ← primera iteración
  4. Calcular el error relativo en x₁: Er = |x₁ − x₀| / |x₁| = |1.66667 − 0| / |1.66667| = 1.00000e+00
  5. Como Er ≥ 1×10⁻⁵, continuar iterando hasta que el error sea menor que la tolerancia. El método converge en 6 iteraciones.

Sobre el método — Secante

El método de la Secante es una variación de Newton-Raphson que no requiere calcular la derivada analítica. En su lugar, aproxima f′(x) usando una diferencia finita con los dos últimos puntos.

Fórmula de iteración:

x_{n+1} = x_n − f(x_n) · (x_n − x_{n−1}) / (f(x_n) − f(x_{n−1}))

La línea que une los puntos (x_{n−1}, f(x_{n−1})) y (x_n, f(x_n)) es la secante. El nuevo punto es donde esa secante cruza el eje x.

Ventajas frente a Newton-Raphson:

  • No necesita derivada analítica
  • Útil cuando f′ es difícil de calcular
  • Convergencia superlineal (orden ≈ 1.618)
  • Solo requiere evaluaciones de f(x)
Procedimiento paso a paso
  1. Definir la función: f(x) = √x − 2·ln(x)
  2. Evaluar en los dos valores iniciales: f(3) = √3 − 2·ln(3) = 1.73205 − 2.19722 = −0.46517 f(5) = √5 − 2·ln(5) = 2.23607 − 3.21888 = −0.98281
  3. Aplicar la fórmula de la Secante con x₀=3, x₁=5: x₂ = 5 − f(5)·(5−3) / (f(5)−f(3)) x₂ = 5 − (−0.98281)·(2) / (−0.98281−(−0.46517)) x₂ = 5 − (−1.96562) / (−0.51764) x₂ = 5 − 3.79731 = 1.20269
    x₂ = 1.20269  ← primera iteración efectiva
  4. Calcular el error relativo: Er = |x₂ − x₁| / |x₂| = |1.20269 − 5| / |1.20269| = 3.15734e+00
  5. Como Er ≥ 1×10⁻⁵, continuar con la ventana deslizante hasta convergencia. El método converge en 9 iteraciones.

Sobre el método — Bisección

El método de Bisección es el más robusto para encontrar raíces. Se basa en el Teorema de Bolzano: si f es continua en [a, b] y f(a)·f(b) < 0, entonces existe al menos una raíz en ese intervalo.

Procedimiento: En cada iteración se evalúa el punto medio c = (a+b)/2. Si f(a)·f(c) < 0, la raíz está en [a, c]; si no, está en [c, b]. El intervalo se reduce a la mitad en cada paso.

c = (a + b) / 2   →   error = |b − a| / 2

Requisito previo: el intervalo inicial debe contener un cambio de signo — es decir, f(a) y f(b) deben tener signos opuestos.

  • Convergencia garantizada
  • No necesita derivada
  • Robusto ante funciones irregulares
  • Fácil de implementar
  • Error acotado en cada paso
Procedimiento paso a paso
  1. Definir la función: f(y) = e⁻ʸ − 4y
  2. Verificar cambio de signo en el intervalo [0.1, 0.3]: f(0.1) = e⁻⁰·¹ − 4(0.1) = 0.90484 − 0.40000 = +0.50484 > 0 f(0.3) = e⁻⁰·³ − 4(0.3) = 0.74082 − 1.20000 = −0.45918 < 0
    f(0.1) · f(0.3) < 0  ← cambio de signo confirmado ✓
  3. Calcular el primer punto medio: c = (a + b) / 2 = (0.1 + 0.3) / 2 = 0.20000
    c₁ = 0.20000  ← iteración 1
  4. Evaluar f(c) y decidir el nuevo subintervalo: f(0.2) = e⁻⁰·² − 4(0.2) = 0.81873 − 0.80000 = +0.01873 Como f(0.1)>0 y f(0.2)>0 → la raíz está en [0.2, 0.3]
  5. Continuar bisectando hasta que el error absoluto sea menor que 0.01. El método converge en 5 iteraciones.

Configuración del método
Newton-Raphson f(t) = cos(t) − e⁻ᵗ = 0 f′(t) = −sin(t) + e⁻ᵗ t₀ = 1 tol = 1×10⁻⁵
Sobre el método — Newton-Raphson

El método de Newton-Raphson construye en cada iteración la recta tangente a la curva f(t) en el punto actual, y usa su intersección con el eje t como siguiente aproximación.

Fórmula de iteración:

t_{n+1} = t_n − f(t_n) / f′(t_n)

Su convergencia es cuadrática: el número de cifras decimales correctas se duplica en cada iteración, por lo que basta con muy pocas iteraciones para alcanzar alta precisión.

  • Convergencia cuadrática
  • Solo 4 iteraciones para tol = 1×10⁻⁵
  • Derivada analítica disponible
  • Error disminuye muy rápidamente
Procedimiento paso a paso
  1. Definir la función y su derivada: f(t) = cos(t) − e⁻ᵗ f′(t) = −sin(t) + e⁻ᵗ
  2. Evaluar en el valor inicial t₀ = 1: f(1) = cos(1) − e⁻¹ = 0.54030 − 0.36788 = 0.17242 f′(1) = −sin(1) + e⁻¹ = −0.84147 + 0.36788 = −0.47359
  3. Aplicar la fórmula de Newton-Raphson para obtener t₁: t₁ = t₀ − f(t₀)/f′(t₀) = 1 − (0.17242)/(−0.47359) t₁ = 1 − (−0.36408) = 1.36408
    t₁ = 1.36408  ← primera iteración
  4. Calcular el error relativo en t₁: Er = |t₁ − t₀| / |t₁| = |1.36408 − 1| / |1.36408| = 2.66903e−01
  5. Como Er ≥ 1×10⁻⁵, continuar iterando. Gracias a la convergencia cuadrática, el método converge en solo 4 iteraciones.
